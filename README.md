# RabbitLabGame

Developed with Unreal Engine 5 (aw hell naw)

## Cheat commands

Open the in-game console and use:

- `SpawnInLevel Level2` — open `/Game/Level2` and spawn at its `PlayerStart`.
- `SpawnInLevel /Game/Level2` — equivalent full package-path form.
- `EnableMeltableDebug`, `DisableMeltableDebug`, or `ToggleMeltableDebug` — control melt-contact debug drawing.
